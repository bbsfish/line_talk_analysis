Abstract

LINEのトーク履歴から、チャットグループ内での発言のデータを集めます。 トーク履歴をテキストで出力できるチャットであれば、基本使えます。 出力の仕方はGUIDEを参照してください。 出力したファイルをInputで選択すれば、 Resultに結果が出力されます。 なお、トーク履歴のファイルはすべてローカルで処理され、データがサーバに 送信・保存されることはありません。

<About column names>

Names:
 User Name.
Message (Text only), Stamp, Photo, Video:
 ユーザに送信されたメッセージ等の数.
Word length:
 ユーザが送信したメッセージの文字数の合計.
length per Msg.:
 ユーザの１メッセージあたりの文字数.
 (計算式)=[Word length] ÷ [Message (Text only)]
